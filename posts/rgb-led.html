RGB LED potentiometer control with Arduino.

This is just a simple RGB LED control project using three potentiometers controlling the R, G, and B values of the LED. I used a common cathode RGB LED and not a common anode.

Check out this video below demonstrating the circut!

In the video, I used a plastic golf ball (used in practice) to diffuse the light. This was not needed, but I would recomend doing so. Without it, the LED show up "blotchy" and certian portions of it are certian colors. It just doesn't mix well without it. Anything really can be used, such as normal ping pong balls, paper (from a slight distance), tissue (also from a slight distance), or anyhting else that is translucent.

You can view the code down below. it is pretty simple. It even is possible to do this project without using an Arduino. With that said, an arduino makes changing the potentiometer rate of change a non-linear rate a lot easier. 

        
          //Pin Setup
          int redPin = 3;
          int bluePin = 6;
          int greenPin = 5;
          int redPot = A5;
          int greenPot = A4;
          int bluePot = A6;

          //Variables
          float conNum = 255. / 1023.;
          int redVal;
          int greenVal;
          int blueVal;

          //Begin code 
          void setup() {
            Serial.begin(9600);
            pinMode(redPin, OUTPUT);
            pinMode(greenPin, OUTPUT);
            pinMode(bluePin, OUTPUT);
            pinMode(redPot, INPUT);
            pinMode(greenPot, INPUT);
            pinMode(bluePot, INPUT);
          }
          void loop() {
            redVal = analogRead(redPot) * conNum;
            greenVal = analogRead(greenPot) * conNum;
            blueVal = analogRead(bluePot) * conNum;
  
            analogWrite(redPin, redVal);
            analogWrite(greenPin, greenVal);
            analogWrite(bluePin, blueVal);

            //Print the RGB Value to the serial monitor
            Serial.print("RGB(");
            Serial.print(redVal);
            Serial.print(",");
            Serial.print(greenVal);
            Serial.print(",");
            Serial.print(blueVal);
            Serial.print(")");
            Serial.println(" ");

          }

Frankly, I know this is a simple project. My use for it is to be able to get a value for a certian color of the LED. Also, it is just fun to mess with the values, and not have to re-upload the code everytime I want to see a new color.
